VerPaste

VerPaste is a simple, distraction-free, note-taking app with sharing, saving and encryption features.

VerPaste does not store any information. The message is either encoded or encrypted (if using a key) within the URL hash and is decoded upon loading the shared URL or decrypted upon providing the key.
